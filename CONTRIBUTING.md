# Contribution Guidelines

We welcome contributions to EO-MASTER! Follow these steps:

## Branch Naming
- **Feature**: `feature/description`
- **Bugfix**: `bugfix/description`

## Pull Request Guidelines
1. Provide a detailed description.
2. Include screenshots for UI changes.
3. Request reviews from at least one team member.

## Coding Standards
- **JavaScript**: Follow ESLint rules.
- **Commits**: Use [Conventional Commits](https://www.conventionalcommits.org).

## Reporting Issues
Open an issue [here](https://github.com/yourusername/EO-MASTER/issues).
